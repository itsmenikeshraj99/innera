# 📚 About

**Innera** is a modern educational mobile application built with **React Native**, **Expo**, **TypeScript**, and **Firebase**.

The app is designed for students of all age groups and provides structured learning, chapter-wise study material, competitive exam preparation, quizzes, and future AI-powered learning assistance.

```bash

cd innera

npm install

npx expo start
```
